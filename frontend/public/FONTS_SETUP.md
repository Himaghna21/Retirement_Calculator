# Font Setup Guide

## Montserrat Font Installation

The Retirement Calculator uses **Montserrat** as the primary brand font, with Arial and Verdana as fallbacks.

### Option 1: Download from Google Fonts (Recommended)

1. Visit: https://fonts.google.com/specimen/Montserrat
2. Download these weights:
   - Regular (400)
   - SemiBold (600)
   - Bold (700)
3. Convert to WOFF2 format (most browsers support it)
4. Place in `public/fonts/` directory
