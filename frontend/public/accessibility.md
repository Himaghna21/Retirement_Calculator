# Accessibility Statement

**Last Updated:** 2026-03-12

## Commitment

HDFC is committed to ensuring digital accessibility of the Retirement Planning Calculator for people with disabilities. We strive to adhere to the most current level of Web Content Accessibility Guidelines (WCAG 2.1) Level AA standards.

## Compliance Status

The Retirement Planning Calculator meets **WCAG 2.1 Level AA** accessibility standards:

- ✓ Perceivable: Content is presented in ways users can perceive
- ✓ Operable: Users can navigate and operate the interface
- ✓ Understandable: Content and interface are clear and understandable
- ✓ Robust: Compatible with current and future assistive technologies

## Features

### Keyboard Navigation
- All interactive elements are accessible via keyboard
- Logical tab order from top to bottom
- Form submission possible with Enter key
- Skip-to-content link for jumping to main content

### Screen Reader Support
- All form labels properly associated with inputs
- Error messages announced clearly
- Results section announced when calculations complete
- Semantic HTML structure for proper document outline
- ARIA labels and descriptions where needed

### Visual Accessibility
- Text contrast meets 4.5:1 ratio (WCAG AA standard)
- Large touch targets (minimum 44x44 pixels)
- Color is never the only indicator of meaning
- Font sizes minimum 14px for disclaimers
- Responsive design works on all screen sizes

### Color & Design
- No reliance on color alone to convey information
- Sufficient contrast for all text and UI elements
- High contrast mode compatible
- Clear focus indicators on all interactive elements

## Testing

We test for accessibility using:

- **Manual Testing:** Keyboard navigation, screen reader testing
- **User Testing:** Feedback from users with disabilities

The Retirement Planning Calculator has been tested and works with:

  - JAWS (Windows)

- **Browser Extensions:**
  - High Contrast
  - Reader Mode
- **Operating System Features:**

  - macOS Dark Mode
