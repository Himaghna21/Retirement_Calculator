# Person 1 — Frontend Developer Setup Guide

## Overview
You are responsible for building the entire Next.js frontend with WCAG 2.1 AA accessibility compliance and strict adherence to HDFC brand guidelines.

## Technology Stack
- **Framework:** Next.js 15.5.9
- **Node.js:** 22.11.0
- **NPM:** 10.9.0
- **Styling:** Vanilla CSS (Brand variables in `styles/variables.css`)
- **Testing:** Jest + React Testing Library
- **Accessibility:** WCAG 2.1 AA Compliance

## Brand Guidelines (Mandatory)
- **Palette**: Blue (#224c87), Red (#da3832), Grey (#919090).
- **Fonts**: Montserrat, Arial, Verdana.
- **Imagery**: Absolutely no growth arrows, currency icons, or exaggerated financial metaphors.

## Getting Started

### 1. Install Dependencies
```bash
npm install
```
